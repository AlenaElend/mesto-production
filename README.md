
# Проект Mesto

Интерактивная страница для добавления и удаления фотографий с возможностью ставить лайки.

## Функциональность

- Редактирование профиля пользователя
- Обновление аватара
- Добавление новых карточек
- Удаление своих карточек
- Постановка и снятие лайков
- Просмотр изображений в полном размере
- Валидация всех форм
- Интеграция с API сервера

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Vite
- Fetch API
- GitHub Pages

## Установка и запуск

### Установка зависимостей
```bash
npm install
```

### Запуск локального сервера разработки
```bash
npm run dev
```

### Сборка проекта
```bash
npm run build
```

### Публикация на GitHub Pages
```bash
npm run deploy
```
